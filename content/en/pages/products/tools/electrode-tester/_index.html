This Electrode resistance tester has been designed to control the continuity of wires / electrodes. The connection of the circuit to be tested is done thanks to the two plugs.

One button is used to read the value of the Ohmmeter, the result will be displayed in Kohm In case of open circuit, no values will be displayed.

The module is supplied by standard cell.
