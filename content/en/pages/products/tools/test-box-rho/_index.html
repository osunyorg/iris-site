This test box is not a calibration unit, but a test unit which aims at checking the good operation of a SYSCAL Switch or a SYSCAL PRO Switch version. It allows to control the Rs Check function and the measuring function of a SYSCAL for each channel.
More precisely, this test box is useful to validate the proper functioning of the internal CPU board, Power board and Switching boards of a SYSCAL/SYSCAL Pro Switch.
