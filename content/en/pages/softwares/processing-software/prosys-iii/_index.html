Data download
Numeric and graphic presentation (raw parameters, resistivity, chargeability,…)
Processing (filter on threshold values, apply a sliding or median average, reject a node, reject the gapfiller quadripoles, create a batch file with a specific filtering to apply to a set of files, …)
Topography insertion
Plot of the apparent resistivity and chargeability sections
Export to various formats (″txt″, Res2dinv, …) and visualization of the exported files

The Prosys III software is a program allowing to transfer, edit, process and export resistivity and chargeability data of the IRIS Instruments resistivity-meters (Syscal) and IP receivers (Elrec).

The main functions of the Prosys III software are the following ones:

  • Data download
  • Numeric and graphic presentation (raw parameters, resistivity, chargeability,…)
  • Processing (filter on threshold values, apply a sliding or median average, reject a node, reject the gapfiller quadripoles, create a batch file with a specific filtering to apply to a set of files, …)
  • Topography insertion
  • Plot of the apparent resistivity and chargeability sections
  • Export to various formats (″txt″, Res2dinv, …) and visualization of the exported files
