Android App to visualize data from Syscal pro / Elrec

body: >-


ProsysView software is an Android App, allowing to edit, process and export resistivity and chargeability data of the IRIS Instruments resistivity-meters (SYSCAL) and IP receivers (ELREC). The main functions of ProsysView are the following ones:
Automatic or manual min / max values to display
Display resistivity pseudo section
Display IP chargeability pseudo section
Display resistivity sounding
Export selected datas
