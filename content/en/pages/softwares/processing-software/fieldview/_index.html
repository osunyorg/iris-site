FieldView is a PC software allowing to retrieve data in real time from a resistivity-meter Syscal Pro / Elrec Pro Switch type.
This allows a real time display of the apparent resistivity pseudo-section and also a possibility in the same time to store the data set in the PC in a file readable by the IRIS Instruments Prosys II software.
So, FieldView allows to have a direct data quality control, in the field, and then, to react quickly in case of the measurements don’t match the user’s requirements.
